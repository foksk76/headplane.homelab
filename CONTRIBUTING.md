# Contributing

Thanks for improving this repository.

## Rules

- Keep `README.md` as the main English source.
- Keep `README.ru.md` aligned with the English README when practical.
- Prefer operationally verified commands over hypothetical examples.
- Document version-specific pitfalls explicitly.

## Suggested workflow

1. Update the English documentation first.
2. Update the Russian translation in the same change when possible.
3. Keep English guides in `docs/` and Russian guides in `docs/ru/` aligned when practical.
4. Add an entry to `CHANGELOG.md` for meaningful documentation updates.
